## Description 📌

This is a project build with **React** and with some other tools like: **Cloudinary** | **React Query** |  **Vite** | **Taildwind** | **Typescript**  | **Nodejs** | **React Router** | **Json Server**  |  **Animate CSS** |  this simulate Funko pop store.
